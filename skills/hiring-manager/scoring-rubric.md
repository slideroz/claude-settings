# Scoring Rubric — Hiring Manager Critique

Use this rubric to produce consistent, calibrated scores across critique sessions. Each dimension is scored on a 4-point scale.

---

## Score Definitions

| Score | Label | Meaning |
|---|---|---|
| 4 | **Strong** | Exceeds expectations for a senior candidate. Would impress a hiring manager. |
| 3 | **Adequate** | Meets the bar. Competent but not differentiated. Passes the screen. |
| 2 | **Needs Work** | Visible gap. Would cause hesitation. Likely to filter on this dimension. |
| 1 | **Missing** | Not present or actively harmful. Fix before submitting to any role. |

---

## Dimension Rubrics

### First Impression

| Score | Description |
|---|---|
| 4 | Title/headline is immediately credible; opening line establishes a clear, differentiated POV; no filler language |
| 3 | Title is clear; summary is competent but generic; identity is present but not sharp |
| 2 | Title is vague or mismatched to target role; summary buries the lead or repeats the title |
| 1 | No clear identity; reads as a list of experiences rather than a coherent practitioner |

---

### Positioning Clarity

| Score | Description |
|---|---|
| 4 | Candidate has a clear, distinct identity that is consistent across all materials; differentiator is legible |
| 3 | Positioning is reasonable but not differentiated; reads as "competent senior designer" without specificity |
| 2 | Positioning is muddled — mixes signals (e.g., freelancer + enterprise + academic at the same time) |
| 1 | No coherent positioning; materials contradict each other or send no clear signal |

---

### Resume Substance

| Score | Description |
|---|---|
| 4 | Bullets demonstrate ownership, judgment, and outcome; systems-level work is visible; scope is legible |
| 3 | Bullets mostly show value but include some activity-language; scope is present; metrics are reasonable |
| 2 | Bullets are primarily task-based ("designed wireframes", "ran user interviews"); value is implied not stated |
| 1 | Resume reads as a job description, not an achievement record; no outcomes, no judgment, no context |

---

### Portfolio / Case Study Signals

| Score | Description |
|---|---|
| 4 | Work demonstrates judgment over output; problems are framed before solutions; constraints are visible; narrative is accessible to non-designers |
| 3 | Work is presented clearly; some problem framing; deliverable-heavy but decisions are present |
| 2 | Portfolio is mostly deliverables; limited narrative; would not hold up in a design critique interview |
| 1 | Portfolio is missing, outdated, or entirely visual with no context or rationale |

---

### Role Fit (when JD is provided)

| Score | Description |
|---|---|
| 4 | Strong match — candidate's signals directly address the role's stated needs; minimal gaps |
| 3 | Moderate match — core competencies align but tailoring needed for emphasis or language |
| 2 | Partial match — candidate has the skills but materials don't communicate fit; significant rewrite needed |
| 1 | Weak match — fundamental misalignment between candidate positioning and role requirements |

---

## Red Flag Definitions

Red flags are specific signals that would cause an experienced hiring manager to stop reading or pass on a candidate. Flag any of the following explicitly in the critique output.

### Credibility Red Flags
- Inflated metrics without context ("increased user satisfaction by 300%")
- Claiming ownership of work that reads as team output without clarification
- Title escalation without corresponding scope escalation
- Unexplained gaps in role history at senior levels

### Positioning Red Flags
- Services language or freelance framing on LinkedIn for a salaried role pursuit
- Portfolio described as "coming soon" or significantly out of date
- Inconsistency between resume title and LinkedIn title
- Website that tries to be a portfolio AND a resume AND a blog without coherent purpose

### Tone Red Flags
- "Passionate about design" or similar filler language
- Excessive hedging ("helped with", "assisted in", "was involved in")
- Academic or theoretical framing without grounding in delivery
- Marketing-speak ("leveraged synergies", "drove alignment", "elevated the brand")

### Content Red Flags
- Every bullet starts with the same verb
- No evidence of receiving or integrating feedback
- No mention of cross-functional collaboration at a senior level
- Tool names as primary resume signal (Figma, Miro, Notion listed prominently without context)

## Hiring Decision Signal — Decision Matrix

Use this at the end of each critique to give an honest read.

| Signal | Criteria |
|---|---|
| **Move to phone screen** | Positioning is clear, materials are credible, differentiation is visible, no major red flags |
| **Move with reservations** | Core competency is there but materials need work; would screen if role is hard to fill |
| **Pass this round** | Materials do not communicate seniority or fit; significant revision needed before applying |
| **Hard pass** | Red flags present that suggest fundamental positioning problem, credibility issue, or misalignment |

Be honest. A "move with reservations" is not a kind alternative to "pass" — it means a specific, recoverable problem exists.
