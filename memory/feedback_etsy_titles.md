name: EtsyShop — KE check before title edits
description: Always run KE on all primary title terms before touching any listing titles
type: feedback
---

Before editing any listing titles, run `mcp__keywords-everywhere__get_keyword_data` on every primary term currently in those titles.

**Why:** Title changes made without verified volume data risk replacing high-performing terms with unvalidated ones. KE data must precede the edit, not follow it.

**How to apply:** Any session that involves listing title work — start with a KE sweep of the current title terms first. No title edits until that data is in hand.
